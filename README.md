# DataStructures-LeetCodePoojaGautamRepo

Welcome to my repository dedicated to mastering Data Structures and Algorithms (DSA) through LeetCode problems! This repository serves as a personal log of my coding journey, showcasing optimized solutions and tracking my progress.

---

##  Repository Goals

*   **Master Core Concepts:** Implement and deeply understand fundamental data structures and algorithmic paradigms.
*   **Continuous Improvement:** Write clean, readable, and highly optimized code (minimizing time and space complexity).
*   **Consistency:** Track daily or regular progress to maintain a strong problem-solving streak.

---

##  Data Structures Covered

The solutions in this repository span a wide variety of topics, including but not limited to:

*   **Linear:** Arrays, Linked Lists, Stacks, Queues
*   **Non-Linear:** Trees, Binary Search Trees (BST), Graphs, Heaps
*   **Advanced Paradigms:** Dynamic Programming (DP), Greedy Algorithms, Recursion & Backtracking, Two Pointers, Sliding Window
